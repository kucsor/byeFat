# **App Name**: byeFat

## Core Features:

- Image Upload/Capture: Allow users to upload images of nutrient labels or capture them using their device's camera.
- Nutritional Data Extraction: Automatically extract nutritional data from the uploaded or captured image using AI, including calories, macros, and serving sizes. The app will use OCR as a tool.
- Manual Data Input: Enable users to manually input nutritional data for products with missing or unreadable information on the label.
- Daily Calorie Log: Display a daily log of calorie intake, showing a breakdown of meals and total calories consumed.
- Responsive Design: Ensure the web app adapts to different screen sizes, including smartphones.

## Style Guidelines:

- Primary color: Fresh Green (#8BC34A), evoking health and vitality, anchoring the design.
- Background color: Off-White (#F5F5F5), for a clean and simple base.
- Accent color: Lime Green (#AED581), an analogous color to the primary green that serves as a contrasting but related highlight, drawing attention to interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif for a modern but slightly warm, easy-to-read feel.
- Use simple, clean icons to represent different food groups and nutrients.
- Employ a clean and intuitive layout for easy navigation and data input on both desktop and mobile devices.
- Use subtle animations to provide feedback during data input and updates to the calorie log.
